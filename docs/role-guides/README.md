## Quick Start

1. Choose your role guide above
2. Follow the setup instructions
3. Try the example workflows
4. Integrate into your daily routine

## Common Workflows

### Requirements → User Stories → Jira Tickets
1. Pull requirements from Confluence
2. Generate user stories with AI
3. Create Jira tickets automatically

### Meeting → Action Items → Jira Tickets
1. Transcribe meeting audio
2. Extract action items with AI
3. Create Jira tickets with owners

### Retrospective → Analysis → Action Items
1. Analyze retrospective comments
2. Identify themes and sentiment
3. Generate action items
4. Create Jira tickets

## Best Practices

1. **Start Small**: Begin with one workflow, then expand
2. **Review AI Output**: Always review and refine AI-generated content
3. **Provide Context**: Include domain knowledge in prompts
4. **Iterate**: Use AI as a starting point, then refine
5. **Integrate**: Connect with your existing tools (Jira, Confluence, etc.)
